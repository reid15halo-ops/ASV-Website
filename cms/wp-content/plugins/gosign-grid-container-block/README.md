## Development
== Description==

### Requirements
This plugin creates single or multiple column blocks on single click. There are number of defined grid block element including single, 
double 50-50, 10-90, 20-80, 30-70, 40-60, 
three column 33-33-33, 11-44-33 and many more.
Every Column also have individual settings including(Background image, color, video, overlay, inner content styling)
This plugin is compatible with Gutenberg.

#### Features:

### Installation
- Creates single or multiple coloums 
- Full width and limited width options
- Add background video
- Add background colors
- Add background images
- Background overlay with opacity
- Inside container styling options
- Custom margins and paddings
- Multiple options to customise inside content elements
- PHP 7.2-support
- Full WordPress 5 (Codename: Gutenberg) support

== Installation ==

All linters compatible with the modern IDE and code editors..
1. Upload \"gosign-grid-container-block\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Add the Gosign - Grid Container Block to the post or page where you want to add beautiful content.
