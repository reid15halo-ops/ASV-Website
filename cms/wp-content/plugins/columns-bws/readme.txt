== Description ==

Columns plugin is a simple solution that helps to add columns with custom content to your WordPress website pages, posts, custom pages, widgets by means of shortcodes. Create and store unlimited column sets with specific settings, customize the appearance, change the content inside each column and paste its shortcode to display the individual column sets wherever you like.

Organize your content into a beautiful layout easily!

[View Demo](https://bestwebsoft.com/demo-columns-by-bestwebsoft/?ref=readme)

== Installation == 

1. Upload the `columns-bws` folder to `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in "Columns".

== Frequently Asked Questions ==

= How can I add Columns to my website? =

1. Create a columns using your WordPress admin panel in "Columns" > "Add New".
2. There are 2 ways to add Columns:
First:
	1) Go to "All Posts";
	2) Choose the post you would like to be displayed by our plugin;
	3) Click "edit" button;
	4) Find "Add BWS Shortcode";
	5) Select columns in the plugins section;
	6) Click "Insert"
	7) Update/Publish the post;
Second:
Go to columns page, copy and paste the shortcode [print_clmns id=#] into your page or post.
